# Server Advertising Manager

Server Advertising Manager is a Windows application designed to help server owners prepare and manage advertisement posts for Facebook groups and Reddit communities.

## Current Version

Version 4.1.0

## Features

- Manage multiple advertisement messages
- Manage advertisement images
- Prepare Facebook group posts
- Prepare Reddit posts
- Use a dedicated Firefox browser profile
- Save application settings between sessions
- Maintain local logs and data
- Portable Windows installation

## Installation

1. Download the latest ZIP file from the Releases section.
2. Right-click the ZIP file and select **Extract All**.
3. Open the extracted `Server Advertising Manager` folder.
4. Run `Server Advertising Manager.exe`.
5. Follow the first-run setup instructions.

Do not run the application directly from inside the ZIP file.

## Browser Login

The application uses its own browser profile. The first time it opens Facebook or Reddit, you may need to sign in manually.

Your login session will then be stored locally inside the application's Browser Profile folder.

## Important Notice

This application assists with preparing advertisements. Users are responsible for following the rules and terms of every website, Facebook group, subreddit, or community where they post.
